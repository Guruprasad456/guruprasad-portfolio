Model Trials

How the research thinking changes from one project domain to another.

Healthcare ML

Reliability + delivery

  • Compare several classical algorithms first
  • Keep the final app practical through a Flask workflow
  • Favor visible end-to-end proof over a single hidden score

Computer Vision

Prototype honestly

  • Use a smaller feature representation and visible state mapping
  • Keep the confusion matrix visible
  • Treat the result as a learning-forward focus coach, not a finished biometric product

Recommendation Systems

Match the model to the dataset

  • Use lighter recommendation logic where the dataset is small
  • Use large-scale feature labelling where data scale supports it
  • Tie the logic to a polished product UI so the output feels useful
